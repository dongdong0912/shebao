# 📌 图1｜退休条件

**标题：退休不是到年龄就行！**

1. 达到法定退休年龄（因人群/政策不同）
2. 养老保险累计缴满15年
3. 医保缴费年限达标（各地不同）
4. 社保记录完整，无断缴

🔴 重点：条件不够，可能影响退休办理

**不是到点退休，是条件达标退休**

---

# 📌 图2｜退休材料

**标题：材料不全＝多跑社保局**

1. 身份证 / 社保卡
2. 户口本 / 证件照
3. 人事档案（关键）
4. 社保缴费记录

🔴 重点：材料不全可能被退回

**材料齐不齐，决定你跑几次窗口**

---

# 📌 图3｜办理流程

**标题：退休4步流程**

1. 提前核对缴费年限
2. 提交退休申请
3. 社保审核材料
4. 次月发养老金

🔴 重点：建议提前1–3个月办理

**提前一步准备，晚年少走弯路**

---

# 📌 图4｜灵活就业

**标题：灵活就业退休要注意**

1. 全靠自己缴费
2. 年限不足需补缴
3. 多地社保需合并
4. 规则因地不同

🔴 重点：定期核对社保记录

**没人替你交社保，你更要盯紧自己**

---

# 📌 图5｜常见坑

**标题：退休最容易踩的坑**

1. 社保断缴未补
2. 异地参保未合并
3. 医保年限不足
4. 档案信息不全
5. 临退休才办理

🔴 重点：问题会影响养老金待遇

**退休不是难，是忽略太久**

---

# 📌 图6｜养老金发放

**标题：养老金发放规则**

1. 审核15–30天（各地不同）
2. 审核通过次月发放
3. 部分地区有补贴

🔴 重点：通过审核才开始发

**养老金不是申请就有，是审核通过才有**
